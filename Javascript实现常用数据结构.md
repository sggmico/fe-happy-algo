# Javascript实现常用数据结构

> 算法往往是基于具体的数据结构来实现的。学习算法之前，有必要全局了解下各种数据结构的实现及相应的复杂度。容易在我们学习算法时扩展思路。

## 1、线性数据结构

### 1.1、数组

**实现：**

```javascript
// 浏览器原生支持
var arr = [1,2,3,4,5]
// 或
var arr = new Array(5)
```

**复杂度分析：**

> 复杂度分为：
>
> - 最好
> - 最差
> - 平均
>
> 下面，给出平均复杂度

| 复杂度 | Access | Insert | Delete |
| ------ | ------ | ------ | ------ |
| Time   | O(1)   | O(n)   | O(n)   |
| Space  | O(n)   | O(n)   | O(n)   |
